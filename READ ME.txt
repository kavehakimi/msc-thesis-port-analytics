# Developing an Intelligent System for Container Terminal Loading and Discharge Operations

MSc Thesis — Bournemouth University (2023)

## Project Overview

This project investigates the application of Machine Learning techniques to improve loading and discharge operations in container terminals. The main objective was to develop an intelligent decision-support system capable of predicting operational patterns and supporting intelligent optimisation of container terminal processes within a real-world port environment.

The research focused on analysing operational container data from Shahid Rajaee Port and applying supervised learning models to support more efficient terminal planning, resource allocation, and container handling operations.

This work was completed as part of the MSc in Data Science and Artificial Intelligence at Bournemouth University.

## Dataset

The dataset used in this project contains sensitive operational data from Shahid Rajaee Port and cannot be shared publicly.

For reproducibility purposes, users may replace the `data/` directory with their own dataset following a similar schema and structure (see notebook comments for guidance).

### Dataset Summary

The dataset contains approximately 20,000 operational container records across multiple container categories, including:

- DC (Dry Container)
- DG (Dangerous Goods)
- FLT (Flat Rack)
- HC (High Cube)
- HRF (High Refrigerated)
- OG (Open Top General)
- OT (Open Top)
- RF (Refrigerated)
- TNK (Tank Containers)

## Model Performance

### Discharge Operation Prediction

Random Forest achieved strong predictive performance on discharge operations:

- Test Accuracy: 84.0%
- Test Precision: 81.7%
- Test Recall: 71.5%
- Test F1-score: 73.8%

### Load Operation Prediction

Random Forest also demonstrated effective prediction capability for loading operations:

- Test Accuracy: 81.6%
- Test Precision: 80.5%
- Test Recall: 83.5%
- Test F1-score: 81.4%
